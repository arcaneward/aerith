aerith.me

Hello and welcome to Aerith.me, a good old fashioned collection of nothingness. This website really serves no purpose, I just wanted to make a website. The domain was purchased through namecheap and hosted on github. Artwork is official renders of Aerith Gainsborough from Final Fantasy VII Remake. Below you can find a few random projects I've made.

/ims - inventory management system written in C# that utilizies a BindingList to keep track of products and associated parts.

/termAdvisor - Andoid app that uses a SQLite database to access and edit academic terms, courses, and evaluations.

/rosterManager - c++ console application that maintains a roster of current students within a given degree program and uses function calls to manipulate data. It parses through the student data and creates objects. The student records are stored in an array and specific student data is output to the console.
